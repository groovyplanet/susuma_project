# 수리 전문가 매칭 플랫폼

- **개발 기간** : `2024/07/03 ~ 07/30` (4주)
- **주요 기능**
   - `사용자`
      - 예약·포인트·프로필 관리 및 채팅
      - 의뢰인 : 수리 예약 및 후기 등록
      - 수리기사 : 수리 예약 승인
   - `관리자`
       - 회원 관리(수리기사 가입 승인)
       - 예약 및 후기 관리
       - 게시판 관리
<br/>

## 🤹‍♂️ 개발 환경 및 기술 스택

| 항목 | 내용 |
|---|---|
| **프로그래밍 언어** | Java 11 |
| **데이터베이스(DB)** | Oracle 11 |
| **애플리케이션 서버** | Apache Tomcat 10 |
| **웹 기술** | ✨**JSP/Servlet**, JSTL |
| **ORM** | MyBatis |
| **API** | Kakao 지도, Daum 우편번호 검색, NAVER 스마트 에디터, PortOne |
| **개발 도구(IDE)** | Eclipse, VSCode |
